# 蓝队实战笔记 (Blue Team Note)

这是一份专注于**网络安全蓝队实战**的经验沉淀笔记。

项目核心主旨在于**“快速发现问题与解决问题”**。摒弃空泛的理论说教，我们致力于通过真实的案例来解析技术细节与实战应用。

例如，在应对典型的应急响应场景时——**“收到通报：内部某设备访问恶意IP/域名”**：
- **精准定位**：如何根据恶意网络指标（IOC），迅速定位至该设备上的可疑进程？本项目总结了三类高效排查路径：基于网络连接状态排查、基于内存特征字符串检索、基于时间线分析溯源。
- **定性研判**：锁定可疑进程后，如何判定其行为性质？本项目同样梳理了三类核心研判手段：静态逆向分析、动态行为调试、云沙箱辅助分析。

除应急响应外，蓝队日常运营还涉及诸多高级实战场景，如**威胁狩猎（Threat Hunting）**、**威胁情报运营**等。本项目当前阶段优先连载“应急处置”相关章节，后续将持续迭代并完善整个蓝队知识体系。
